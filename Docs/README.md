# wnt5a regulation mechanism exploration

## Introduction

Mutations in WNT5A and other WNT5A signaling molecules lead to Robinow syndrome. This topic is widely studied. However there is no information on how WNT5A is regulated.

In zebrafish, a subpopoulation of wnt5a-positive dermal fibroblasts was identified. In this project, two specific ATAC peaks in front of the wnt5a gene are disected to find what is regulationg the wnt5a expression.
